# Fonts

Drop any `.ttf` / `.otf` file here and reference it from `configs/variants.yaml` via the `font` key (or leave `font: null` to use PIL's default bitmap font).

Recommended free fonts for ad overlays:

- **Inter** — https://fonts.google.com/specimen/Inter (clean sans, good for Minimal / Urban variants)
- **Playfair Display** — https://fonts.google.com/specimen/Playfair+Display (serif, good for Luxury)
- **Lora** — https://fonts.google.com/specimen/Lora (warm serif, good for Nostalgic)

Fonts are **not committed** to the repo (see `.gitignore`). Each user downloads them locally under their chosen license.
